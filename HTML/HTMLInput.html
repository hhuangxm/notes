HTML输入类型

平时使用input标签的时候，也只是知道<input type="text">这样使用input标签，但是HTML5却增加了很多input类型：

输入类型：text

这个应该是最常用的类型了，只要有表单输入，基本都会用到的类型

实例：

	<form>
		First name:<br>
		<input type="text" name="firstname">
		<br>
	 	Last name:<br>
		<input type="text" name="lastname">
	</form>

Last name:

输入类型：password

输入一些保密信息，如用户密码等时，使用password，这个也算是比较常见的类型

实例：

	<form>
		User name:<br>
		<input type="text" name="username">
		<br>
	 	User password:<br>
		<input type="password" name="psw">
	</form>

显示效果：

User name:

User password:

输入类型：submit

这个也是常用的类型，提交表单处理程序的按钮

实例：

	 	Last name:<br>
		<input type="text" name="lastname">

输入类型： radio

单选按钮，也都知道，比较常用

实例：

	<form>
		<input type="radio" name="sex" value="male" checked>Male
		<br>
		<input type="radio" name="sex" value="female">Female
	</form>

显示效果：

Male
Female

输入类型： checkbox

接下来肯定是复选按钮，也都知道，比较常用

实例：

	<form>

		<input type="checkbox" name="vehicle" value="Car">I have a car 
	</form>

显示效果：

I have a bike

输入类型： datetime，datetime-local

这两个都是有时分秒的日期类型，不同的是datetime的是有时区的，而datetime-local是没有时区的，就这么简单的区别，用法什么的也都一样，就不一一举例子了

实例：

	<form>
		Birthday (date and time):
		<input type="datetime" name="bday">
	</form>

Birthday (date and time):

输入类型：month, week, time,

这三种类型的输入框就不介绍了，简单说一下概念就好，用法都一样，只是功能有一些稍微的差异，month,允许用户选择月份和年份。weebk,允许用户选择周和年。time,允许用户选择时间（无时区）。

当然是输入数字类型的啦，这还用说，而且还可以配合min,max属性，将数字大小控制在一个范围内，当然text类型也可以控制字符串的长度，使用的是minlength,maxlength
