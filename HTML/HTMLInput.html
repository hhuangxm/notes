HTML输入类型

平时使用input标签的时候，也只是知道<input type="text">这样使用input标签，但是HTML5却增加了很多input类型：

输入类型：text

这个应该是最常用的类型了，只要有表单输入，基本都会用到的类型

实例：

	<form>
		First name:<br>
		<input type="text" name="firstname">
		<br>
	 	Last name:<br>
		<input type="text" name="lastname">
	</form>

显示效果：

First name:

Last name:

输入类型：password

输入一些保密信息，如用户密码等时，使用password，这个也算是比较常见的类型

实例：

	<form>
		User name:<br>
		<input type="text" name="username">
		<br>
	 	User password:<br>
		<input type="password" name="psw">
	</form>

显示效果：

User name:

User password:

输入类型：submit

这个也是常用的类型，提交表单处理程序的按钮

实例：

	<form>
		First name:<br>
		<input type="text" name="firstname">
		<br>
	 	Last name:<br>
		<input type="text" name="lastname">
		<br>
		<input type="submit" value="Submit">
	</form>

显示效果：

First name:

Last name:

输入类型： radio

单选按钮，也都知道，比较常用

实例：

	<form>
		<input type="radio" name="sex" value="male" checked>Male
		<br>
		<input type="radio" name="sex" value="female">Female
	</form>

显示效果：

Male
Female

输入类型： checkbox

接下来肯定是复选按钮，也都知道，比较常用

实例：

	<form>
		<input type="checkbox" name="vehicle" value="Bike">I have a bike
		<br>
		<input type="checkbox" name="vehicle" value="Car">I have a car 
	</form>

显示效果：

I have a bike
I have a car

输入类型： button

个人不经常用这个，因为可以使用button标签代替

实例：

	<form>
		<input type="button" onclick="alert('Hello World!')" value="Click Me!">
	</form>

输入类型： color

这些类型，一看名字就知道意思，<input type="color">用于应该包含颜色的输入字段。这个很有意思呢，获取到用户输入的颜色之后，可以根据用户输入的颜色改变主题的颜色，很实用呢！越来越有意思了

实例：

	<form>
		<input type="color" name="favcolor">
	</form>

显示效果：

Select your favorite color:
